Info

The Matlab and Working Model files can be used to perform the position, velocity, and acceleration analyses of a four-bar linkage and to check Homework Problems 4.6, 4.7, 6.4, 6.6, 7.3, and 7.4 of the textbook. All parameters are defined in Figures P4.1, P6.1, and P7.1 respectively.

To change the dimensions of the Working Model file, simply drag the slider control to change the link length or the orientation angle. If the link length is beyond the maximum/minimum of the slider range, adjust the range in the property table.

Source

This Working Model file is adapted from Figure P4.1 on p.198 in Design of Machinery, 3rd ed. by Norton, R.L., McGraw-Hill, 2004.
